How to play

  • Guess the color combination
  • The combination can have any number of each colour as well as blanks
  • Click a button to bring up a colour selection wheel. Click one of the colours to choose that colour.
  • To select a blank button, either leave the button blank or click on a coloured button to reset it.
  • Click the button next to the row to end the turn
  • After each turn, the pins will fill in either white or yellow
  • A yellow pin indicates a correct colour in the correct place
  • A white pin indicates a correct colour in the incorrect place
  • The order of the pins does not reflect the order of the choices
